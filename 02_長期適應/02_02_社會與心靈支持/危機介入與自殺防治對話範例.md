# 危機介入與自殺防治對話範例

## 概述

本指南基於危機介入理論、自殺防治最佳實務及台灣本土文化考量，提供危機狀況下的專業對話範例。運用SLAP模型、QPR技巧等實證方法，協助在緊急心理狀況下提供適當支持。

---

## 🚨 自殺風險評估對話

### 基於QPR (Question, Persuade, Refer) 模型

#### 情境：發現自殺警訊的初步接觸

**警訊識別後的開場對話：**

```yaml
直接但溫和的詢問：
支持者：「我注意到你最近說了一些讓我擔心的話，比如『活著沒意思』。我想直接問你：你有想過要結束自己的生命嗎？」

個案：「有時候會想...但我不確定我真的會去做。」

支持者：「謝謝你的誠實，這需要很大的勇氣。有這樣的想法一定很痛苦。你願意告訴我這些想法是什麼時候開始的嗎？」

具體性評估：
支持者：「當你想到要結束生命時，你有想過用什麼方式嗎？」

個案：「我想過吃藥...但我沒有準備什麼藥。」

支持者：「我理解。聽起來這個想法有時候會出現，但你還沒有具體的計畫。這很重要。那麼，當這些想法出現時，有什麼事情會讓你停下來，不去行動？」

個案：「我會想到我媽媽，她會很傷心...」

支持者：「媽媽對你很重要。你對她的愛，還有她對你的愛，就像一條invisible（看不見的）lifeline（救生索）。雖然現在很痛苦，但這份愛還在那裡對嗎？」
```

#### 情境：高風險個案的緊急介入

**immediate safety planning（立即安全計畫）對話：**

```yaml
緊急安全評估：
支持者：「聽起來你現在非常痛苦，而且有具體想要結束生命的想法。這讓我很擔心你的安全。我需要知道：你現在有take action（採取行動）的衝動嗎？」

個案：「我覺得我快要控制不住了...」

支持者：「謝謝你告訴我。現在我們需要確保你是安全的。首先，你現在身邊有任何可能傷害自己的東西嗎？」

環境安全化：
支持者：「我們可以一起把這些東西temporary（暫時）移開嗎？這不是因為我不信任你，而是要讓你的environment（環境）支持你想要活下去的那一部分。」

支持網絡啟動：
支持者：「現在我們來想想誰可以陪伴你。有沒有一個人，當你call（打電話）他們時，他們會immediately（立即）come（來）陪你？」

個案：「我姊姊...但我不想讓她擔心。」

支持者：「我理解你不想burden（負擔）她。但如果roles reversed（角色對調），如果是她需要help（幫助），你會希望她告訴你嗎？」

個案：「會...我會想要知道。」

支持者：「那我們現在可以call（打電話）給她嗎？我可以陪你一起說。」

專業資源連結：
支持者：「除了姊姊的support（支持），我們也需要professional help（專業幫助）。我想安排你今天就去看醫生，不是因為你crazy（瘋了），而是因為你的心現在很hurt（受傷），需要professional care（專業照顧）。」
```

## ⚠️ 危機狀態對話技巧

### 基於SLAP模型 (Stabilize, Listen, Assess, Plan)

#### 情境：急性精神症狀的危機介入

**情境設定**：個案出現被害妄想，認為有人要傷害他

**Stabilize（穩定化）階段對話：**

```yaml
環境穩定：
支持者：「我可以看出你現在很害怕，覺得不安全。我想幫助你。首先，我們找個quiet（安靜）的地方坐下來好嗎？」

個案：「他們在監視我，到處都有攝影機...」

支持者：「聽起來你感到被watch（監視），這一定很frightening（可怕）。現在你跟我在這個room（房間）裡，我沒有看到攝影機，但我相信你的fear（恐懼）是real（真實的）。」

情緒穩定：
支持者：「你的breathing（呼吸）很快，heart（心）一定跳得很快。我們一起試試slow down（慢下來）好嗎？跟我一起：inhale（吸氣）...hold（憋住）...exhale（呼氣）...」

現實錨定：
支持者：「現在告訴我，你能感覺到這張chair（椅子）嗎？它是什麼material（材質）？」

個案：「是...是塑膠的。」

支持者：「很好。你能聽到外面的traffic（交通）聲音嗎？這些sounds（聲音）tell us（告訴我們）我們在real world（真實世界）裡，現在這個moment（時刻）。」
```

**Listen（傾聽）階段對話：**

```yaml
非評判傾聽：
支持者：「你剛才說『他們』在監視你。這對你來說一定很overwhelming（令人不知所措）。你願意告訴我更多嗎？」

個案：「從昨天開始，我就覺得有人在follow（跟蹤）我，我的電話也被tap（竊聽）了。」

支持者：「這sounds（聽起來）很scary（可怕）。被跟蹤和竊聽會讓任何人都feel（感到）paranoid（偏執）。你最擔心什麼會happen（發生）？」

情緒驗證：
個案：「我怕他們會hurt（傷害）我的家人。」

支持者：「保護家人的想法shows（顯示）你有多愛他們。這種protective（保護性的）feeling（感覺）很natural（自然）。現在你的家人在哪裡？他們safe（安全）嗎？」

經驗探索：
支持者：「這種feeling（感覺）是最近才開始的，還是之前也有過？」

個案：「以前沒有...是這幾天才開始的。」

支持者：「那這幾天有發生什麼special（特別）的事情嗎？工作、sleep（睡眠）、或其他stress（壓力）？」
```

**Assess（評估）階段對話：**

```yaml
功能評估：
支持者：「這些experiences（經驗）對你的daily life（日常生活）有什麼impact（影響）？你能eat（吃飯）、sleep（睡覺）嗎？」

個案：「我已經兩天沒有好好睡了，一直在watch（觀察）窗戶。」

支持者：「沒有sleep（睡眠）會讓everything（一切）更difficult（困難）。你的body（身體）和mind（心智）都很tired（疲累）對嗎？」

insight評估：
支持者：「你覺得這些experiences（經驗）對其他人來說normal（正常）嗎？」

個案：「我...我不確定。也許是我too sensitive（太敏感）了？」

支持者：「懷疑自己的perceptions（感知）需要很大的courage（勇氣）。這shows（顯示）你還有很好的judgment（判斷力）。」

支持系統評估：
支持者：「除了family（家人），還有誰是你trust（信任）的？如果你告訴他們這些experiences（經驗），你覺得他們會怎麼react（反應）？」
```

**Plan（計畫）階段對話：**

```yaml
即時計畫：
支持者：「我們需要讓你feel more safe（感到更安全），也需要讓你的mind（心智）get some rest（得到休息）。我建議我們today（今天）就去see（看）醫生。」

個案：「你覺得我crazy（瘋）了嗎？」

支持者：「絕對不是。你的experiences（經驗）是real（真實的），但你的brain（大腦）現在很tired（疲累），可能需要一些help（幫助）來rest（休息）。就像body（身體）生病要看醫生一樣。」

長期計畫：
支持者：「醫生可能會給你一些medication（藥物）幫助你sleep（睡眠）和feel calmer（感到更平靜）。同時，我們也要make sure（確保）你有people（人）可以check on（關心）你。」

個案：「如果這些feelings（感覺）又come back（回來）怎麼辦？」

支持者：「如果這些feelings（感覺）回來，你可以：1. Call（打電話）給我或醫生，2. 做我們today（今天）練習的breathing（呼吸），3. 提醒自己這是temporary（暫時的），會pass（過去）的。」
```

## 🔥 急性焦慮與恐慌發作對話

### 基於認知行為治療的危機介入

#### 情境：恐慌發作的即時處理

**恐慌發作進行中的對話：**

```yaml
即時穩定技巧：
支持者：「我看到你在hyperventilating（過度換氣），heart racing（心跳加速）。這是panic attack（恐慌發作），雖然很scary（可怕），但不會hurt（傷害）你。」

個案：「我...我覺得我要die（死）了...喘不過氣...」

支持者：「你不會死的。你的body（身體）現在在『fight or flight』（戰鬥或逃跑）mode（模式），就像alarm system（警報系統）誤報一樣。我們來reset（重置）這個alarm（警報）。」

呼吸技巧引導：
支持者：「跟我一起breathe（呼吸）。我們用belly（肚子）呼吸，不是chest（胸部）。把hand（手）放在肚子上，感覺它rise（上升）和fall（下降）。」

個案：「我做不到...太難了...」

支持者：「沒關係，很多人第一次都覺得difficult（困難）。我們試試更簡單的：4秒吸氣，6秒呼氣。Imagine（想像）你在blow（吹）一個很大的bubble（泡泡）。」

接地技巧應用：
支持者：「現在我們用5-4-3-2-1技巧。告訴我你能see（看到）的5樣東西。」

個案：「椅子...桌子...你的衣服...窗戶...」

支持者：「很好！現在4個sounds（聲音）。」

個案：「空調聲...外面的車聲...我的heartbeat（心跳）...你的voice（聲音）。」

支持者：「Perfect（完美）！你的mind（心智）正在comeback（回來）到present moment（當下）。」

認知重構：
支持者：「恐慌讓你的mind（心智）告訴你很多scary（可怕）的things（事情）。但facts（事實）是：你是safe（安全的），你在breathing（呼吸），你的heart（心）在healthy（健康地）beating（跳動）。」

個案：「但感覺還是很terrible（糟糕）...」

支持者：「Feelings（感覺）和facts（事實）是different（不同的）。就像watching（看）恐怖電影會scared（害怕），但你知道你actually（實際上）是safe（安全的）。你的feelings（感覺）valid（有效），但they don't（它們不）define（定義）reality（現實）。」
```

#### 情境：創傷觸發的急性反應

**flashback或dissociation的處理：**

```yaml
現實定向：
支持者：「我notice（注意到）你的eyes（眼睛）看起來very far away（很遙遠）。你還能hear（聽到）我的voice（聲音）嗎？」

個案：（沒有回應，看起來恍惚）

支持者：「如果你能hear（聽到）我，squeeze（捏）我的hand（手）。你現在在2024年，在這個safe（安全的）room（房間）裡，跟我在一起。」

grounding techniques（接地技巧）：
支持者：「我們來做一些physical（身體的）grounding（接地）。Feel（感覺）你的feet（腳）在floor（地板）上。Press（按壓）them down（向下），feel（感覺）the solid（堅實的）ground（地面）。」

個案：（開始有反應）「我...我在哪裡？」

支持者：「你在safe place（安全的地方），跟我在一起。剛才你的mind（心智）travel（旅行）到past（過去）了，但現在你back（回來）了。」

trauma-informed response（創傷知情回應）：
支持者：「剛才發生的是very normal（很正常的）trauma response（創傷反應）。你的brain（大腦）有時候會confuse（混淆）past（過去）和present（現在），但這不是your fault（你的錯）。」

個案：「我感覺像我又回到那個時候...」

支持者：「那個time（時間）已經過去了。你survived（倖存了），你現在safe（安全）。你的body（身體）記住了danger（危險），但你的mind（心智）knows（知道）你現在protected（被保護著）。」

安全重建：
支持者：「我們來remind（提醒）你的system（系統）現在是safe（安全的）。Look around（環顧四周），tell（告訴）me what makes（讓）this place（地方）safe（安全）。」

個案：「有door（門）...我可以leave（離開）...你在這裡...」

支持者：「Exactly（正確）！你有choices（選擇），你有control（控制權），你不是alone（孤單的）。這些都是present moment（當下）的gifts（禮物）。」
```

## 🏠 家庭危機介入對話

### 基於家庭系統理論的危機處理

#### 情境：家庭暴力危機

**安全第一的評估對話：**

```yaml
immediate safety assessment（立即安全評估）：
支持者：「我看到你有一些injuries（傷口），我很擔心你的safety（安全）。現在那個hurt（傷害）你的人在哪裡？」

受害者：「他出去了...但他說會回來...」

支持者：「我們需要確保你是safe（安全的）。你有somewhere（某個地方）可以go（去）嗎？朋友、family（家人）、或者你知道的shelter（庇護所）？」

受害者：「我不能離開，孩子們還在sleeping（睡覺）...」

支持者：「保護孩子們是very important（非常重要），但保護你also（也）是保護他們。如果你不safe（安全），你就不能protect（保護）他們。」

empowerment dialogue（賦權對話）：
支持者：「這不是your fault（你的錯）。沒有人deserves（應得）被hurt（傷害），regardless of（無論）what happened（發生了什麼）。」

受害者：「但我應該be a better wife（做個更好的妻子）...」

支持者：「Being a good wife（好妻子）不包括being a punching bag（沙袋）。Love（愛）不會hurt（傷害）。Real love（真正的愛）protects（保護），不是destroys（摧毀）。」

options exploration（選項探索）：
支持者：「我們來看看你的options（選項）。你可以：1. Call police（報警），2. Go to shelter（去庇護所），3. Stay with trusted friend（住在信任的朋友家），4. Get restraining order（申請保護令）。哪個感覺most possible（最可能）？」

受害者：「我害怕他會更angry（生氣）...」

支持者：「Fear（恐懼）是understandable（可以理解的）。但staying（留下）also（也）has risks（風險）。We can make safety plan（我們可以做安全計畫）for any choice（任何選擇）you make（你做的）。」
```

#### 情境：青少年離家出走危機

**family communication facilitation（家庭溝通促進）：**

```yaml
多方perspective gathering（多方觀點收集）：
調解者：「我們今天聚在這裡是因為everyone（每個人）都在pain（痛苦中）。小明離家讓大家都很worried（擔心），但他也一定有his reasons（他的理由）。」

對父母：
調解者：「做parents（父母）一定很scary（可怕），不知道孩子在哪裡，是否safe（安全）。你們最worried（擔心）什麼？」

父親：「我們怕他學壞，交到bad friends（壞朋友）。」

調解者：「保護孩子是parent instinct（父母本能）。這shows（顯示）你們多麼愛他。」

對青少年：
調解者：「小明，離開家一定也很difficult（困難）。你願意告訴我們what happened（發生了什麼）讓你覺得必須leave（離開）？」

小明：「他們完全不listen（聽）我說話，只會command（命令）我。」

調解者：「Feeling unheard（感到不被聽見）很frustrating（令人沮喪）。你希望parents（父母）understand（理解）什麼？」

bridge building（橋樑建立）：
調解者：「我聽到兩邊都有很deep（深刻的）love（愛），但also（也有）很deep（深刻的）pain（痛苦）。爸媽的protection（保護）comes from（來自）love（愛），小明的need for independence（獨立需求）也是healthy（健康的）growing up（成長）。」

對父母：
調解者：「也許我們可以think about（想想）different ways（不同方式）to protect（保護）小明，ways that make him feel trusted（被信任）rather than controlled（被控制）？」

對小明：
調解者：「也許你可以help（幫助）爸媽understand（理解）how to love you（愛你）in a way that doesn't feel suffocating（令人窒息）？」

collaborative problem solving（合作解決問題）：
調解者：「我們來make（做）一個experiment（實驗）。下個月，我們try（嘗試）new rules（新規則）：more freedom（更多自由）but also（但也要）more communication（更多溝通）。小明gets（得到）later curfew（更晚的宵禁），但needs to（需要）text（傳簡訊）every few hours（每幾個小時）. How does that sound（聽起來如何）？」
```

## 🏥 醫療危機心理支持對話

### 基於醫療創傷知情照護

#### 情境：突然診斷重病的患者

**diagnosis processing support（診斷處理支持）：**

```yaml
shock acknowledgment（震驚承認）：
支持者：「剛才醫生告訴你diagnosis（診斷），我imagine（想像）這是overwhelming（令人不知所措的）news（消息）。」

患者：「我不敢相信...昨天我還好好的...」

支持者：「這種shock（震驚）是completely normal（完全正常的）。我們的mind（心智）需要time（時間）來process（處理）life-changing news（改變生活的消息）。」

information processing（資訊處理）：
患者：「醫生說了很多medical terms（醫學術語），我都沒聽懂...」

支持者：「Medical information（醫療資訊）can be（可能很）overwhelming（令人不知所措），especially when（特別是當）you're in shock（震驚中）. 我們可以together（一起）write down（寫下）questions（問題）for next time（下次）見醫生。」

hope and realism balance（希望與現實平衡）：
患者：「我是不是要die（死）了？」

支持者：「Having（有）serious illness（嚴重疾病）doesn't mean（不意味著）immediate death（立即死亡）。Modern medicine（現代醫學）has many tools（工具），而且你're not alone（你不孤單）in this fight（戰鬥）。」

support network activation（支持網絡啟動）：
支持者：「這不是你一個人的battle（戰鬥）。誰是你most important（最重要的）support people（支持的人）？我們怎麼讓他們know（知道）你need them（需要他們）？」

患者：「我不想worry（擔心）我的family（家人）...」

支持者：「Protecting people（保護人們）we love（我們愛的）is natural（自然的），但sometimes（有時候）sharing（分享）our burden（負擔）helps（幫助）everyone（每個人）feel closer（更親近）and more useful（更有用）。」
```

#### 情境：親人突然死亡的家屬

**acute grief support（急性哀傷支持）：**

```yaml
immediate presence（立即陪伴）：
支持者：（安靜地坐在旁邊，不說話，準備紙巾和水）

家屬：「這不可能...他昨天還跟我說話...」

支持者：「我知道這很difficult（困難）to believe（相信）。Sudden loss（突然失去）feels（感覺）impossible（不可能）to accept（接受）。」

practical support（實務支持）：
家屬：「我不知道該做什麼...有這麼多things（事情）要處理...」

支持者：「Right now（現在）你不需要think about（想）all the things（所有事情）. 我們focus on（專注於）just（只是）the next hour（下個小時）. Do you need（你需要）to call（打電話給）anyone（任何人）? Do you want（你想要）me to help（幫忙）with any calls（任何電話）?」

grief normalization（哀傷正常化）：
家屬：「我一直在cry（哭），但有時候又感覺nothing（什麼都沒感覺）...我是不是weird（奇怪）？」

memory honoring（記憶尊崇）：
支持者：「當你ready（準備好）時，也許你可以tell（告訴）me about（關於）他。What would（什麼會）make him smile（讓他微笑）if he could hear（聽到）us talking（談話）?」
```

**telephone crisis counseling（電話危機諮詢）技巧：**

```yaml

來電者：「沒有人care（關心）我...我覺得我disappear（消失）大家都不會notice（注意到）...」

諮詢員：「That sounds（聽起來）incredibly lonely（非常孤單）. 但你called（打電話）tonight（今晚），這tells（告訴）me part of you（你的一部分）還想要connection（連結）. 你打電話給我們就是reaching out（伸出手）的brave（勇敢）act（行為）。」

resource mobilization（資源動員）:
諮詢員：「You mentioned（你提到）feeling like no one cares（沒人關心）. 但in your whole life（整個生活），was there ever（曾經有過）anyone（任何人）who did care（關心）about you（你）? Maybe（也許）someone（某人）from your past（過去）?」

諮詢員：「She must have been（她一定是）very special（很特別）. What would（什麼會）she say（她會說）to you right now（現在）if she could hear（聽到）us talking（談話）?」

hope instillation（注入希望）:
來電者：「She would say（她會說）'never give up（永不放棄）'...」

諮詢員：「That sounds like（聽起來像）her love（她的愛）speaking through（透過）you. Maybe（也許）part of（一部分）keeping her memory alive（讓她的記憶活著）is honoring（尊重）what she taught（教）you about perseverance（堅持）?」
```
