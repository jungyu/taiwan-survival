# 偏鄉山區避難時限規劃

## 概述

偏鄉山區具有天然屏障優勢，軍事價值較低，自然資源豐富，但交通不便、醫療教育資源有限。本指南依據避難時限分為立即避難、短期避難、長期避難三個層次，提供偏鄉山區居民及外來避難者在不同威脅情境下的最佳避難策略。

---

## ⚡ 立即避難（遇空襲或駁火）

### 威脅情境分析

```
空襲威脅：
• 低空飛行器攻擊
• 山區軍事設施遭攻擊
• 交通要道封鎖
• 通訊設施被破壞

地面威脅：
• 山區據點爭奪
• 重要隘口控制
• 原住民部落衝突
• 非法武裝入侵
```

### 立即避難原則

```
時間限制：警報後10-15分鐘內完成
行動原則：
• 利用天然地形掩護
• 進入山洞、岩穴、密林
• 避免在稜線、山頂暴露
• 快速通知部落族人
```

### 花蓮縣山區立即避難範例

#### 範例一：秀林鄉太魯閣地區

```
情境：白天觀光時間，中橫公路遭攻擊

立即避難選擇：
優先順序1：太魯閣峽谷天然洞穴
• 距離：步行5-10分鐘可達
• 優勢：天然岩石掩護，防護力極強
• 容量：各洞穴可容納20-100人
• 水源：立霧溪水源充足

優先順序2：布洛灣台地原住民聚落
• 距離：步行15-20分鐘
• 優勢：太魯閣族部落，熟悉地形
• 資源：部落有基本生活設施
• 管理：族人自主管理

優先順序3：山月吊橋下方河床
• 距離：步行10分鐘
• 優勢：河谷地形，不易被發現
• 限制：季節性水位變化

避免區域：
• 中橫公路主線（交通目標）
• 太魯閣遊客中心（人群聚集）
• 山頂觀景台（暴露位置）
• 主要步道入口（易被監控）
```

## 🏃‍♂️ 短期避難（24-72小時）

### 威脅情境分析

```
軍事控制：
• 山區據點設立
• 交通要道管制
• 通訊監控加強
• 物資供應中斷

社會動盪：
• 平地難民湧入
• 原住民與漢人衝突
• 資源競爭激烈
• 部落內部分歧
```

### 短期避難原則

```
時間規劃：3-7天避難準備
生存策略：
• 利用傳統狩獵採集知識
• 建立臨時庇護所
• 組織部落防禦體系
• 確保水源食物供應
```
