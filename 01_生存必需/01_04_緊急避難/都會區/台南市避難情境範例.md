# 台南市避難情境範例

## 概述

台南市為台灣古都，擁有豐富歷史文化資產，也是台灣重要的工業基地。地理位置從海岸平原延伸至山區，具有多樣化的避難環境選擇。本文件針對台南市各區域特性，提供具體的避難情境範例。

---

## ⚡ 立即避難情境範例

### 中西區歷史古都

#### 情境一：台南市政府周邊
```
威脅：台南政治中心遭攻擊
地點：台南市政府、中西區
人員：市府員工、洽公民眾、觀光客

立即避難選擇：
優先順序1：台南市政府地下停車場
• 距離：步行1-2分鐘
• 優勢：政府建築結構堅固
• 容量：大型地下空間
• 管理：市府保全管理

優先順序2：成功大學醫學院地下室
• 距離：步行5分鐘
• 優勢：醫學院建築安全標準高
• 設施：醫療資源充足
• 管理：大學專業管理

優先順序3：赤嵌樓文化園區建築
• 距離：步行8分鐘
• 優勢：古蹟建築歷史悠久
• 文化：重要文化資產
• 限制：古建築空間有限

古都特殊考量：
• 古蹟建築結構需特別評估
• 觀光客多，語言溝通問題
• 巷弄狹窄，疏散路線有限
• 文化資產保護重要性

避免區域：
• 台南市政府大樓（政治目標）
• 安平古堡（重要古蹟，人群聚集）
• 孔廟周邊（文化敏感區域）
• 中正路商圈（人群密集）
```

#### 情境二：台南火車站商圈
```
威脅：台南車站交通樞紐遭攻擊
地點：台南火車站、北區商圈
人員：通勤族、購物民眾、旅客

立即避難選擇：
優先順序1：台南火車站地下空間
• 距離：就近可達
• 優勢：火車站地下空間廣闊
• 容量：可容納數千人
• 設施：車站基本設施

優先順序2：成功大學光復校區
• 距離：步行10分鐘
• 優勢：大學建築結構安全
• 容量：校園空間廣大
• 管理：大學保全管理

優先順序3：台南文化中心地下室
• 距離：步行15分鐘
• 優勢：文化建築結構良好
• 設施：完整文化設施
• 管理：文化局管理

避難注意事項：
• 台南車站為南部重要交通樞紐
• 成功大學校園廣大，疏散空間充足
• 北區商圈人潮密集
• 多條道路可快速疏散

避免區域：
• 台南火車站月台區（交通目標）
• 北區商圈密集區（人群聚集）
• 成功路主要幹道（交通要道）
```

### 安南區工業港區

#### 情境三：台南科技園區
```
威脅：科技園區遭攻擊或化學災害

• 管理：學校保全管理
• 環境：遠離工業污染源

優先順序2：台江國家公園管理處
• 距離：步行15分鐘
• 優勢：國家公園建築堅固
• 環境：自然環境優美
• 管理：國家公園管理

優先順序3：社區活動中心
• 距離：就近選擇
• 優勢：社區公共設施
• 管理：社區自主管理

科技園區特殊風險：
• 半導體廠房化學物質
• 精密機械設備
• 高科技產業機密
• 外籍技術人員

化災應變措施：
□ 注意風向避開下風處

• 科技園區廠房密集區

## 🏃‍♂️ 短期避難情境範例

### 玉井、楠西山區

#### 情境五：都會區往山區避難
```

威脅：台南都會區軍事威脅

目標：山區短期避難3-7天

避難路線：
主要路線：台南→玉井→楠西
起點：台南市區
↓ 沿台20線東行
玉井區
↓ 沿南188線北上
楠西區
↓ 往曾文水庫方向
南化區

距離：約50公里
時間：開車1.5小時，大眾運輸3小時

避難設施選擇：
優先選擇：玉井國小、楠西國小
• 山區學校設施
• 可容納數百人

• 農村社區環境

備用選擇：曾文水庫風景區
• 水庫周邊設施
• 自然環境優美
• 水源充足
• 需注意水庫安全管制

山區避難特色：
• 玉井芒果之鄉
• 楠西梅嶺風景區
• 曾文水庫水源
• 西拉雅族文化

避難資源：
□ 曾文溪水系水源
□ 山區農業資源
□ 果樹栽培技術
□ 西拉雅族文化學習

避免區域：
• 曾文水庫大壩周邊
• 台20線主要路段

• 大型觀光景點
