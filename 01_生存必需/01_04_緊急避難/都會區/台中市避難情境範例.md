# 台中市避難情境範例

## 概述

台中市位於台灣中部，為中台灣政治經濟中心，擁有清泉崗機場、台中港、科技園區等重要設施。地形從海岸平原延伸至中央山脈，提供多樣化避難選擇。本文件針對台中市各區域特性，提供具體的避難情境範例。

---

## ⚡ 立即避難情境範例

### 西屯區七期重劃區

#### 情境一：台中市政府周邊
```
威脅：台中政治經濟中心遭攻擊
地點：台中市政府、七期重劃區
人員：市府員工、金融業人士、購物民眾

立即避難選擇：
優先順序1：台中市政府地下停車場
• 距離：步行1-2分鐘
• 優勢：政府建築結構堅固
• 容量：大型地下空間
• 管理：市府保全管理

優先順序2：新光三越台中店地下樓層
• 距離：步行3分鐘
• 優勢：百貨公司地下空間大
• 設施：商場基本設施
• 管理：專業物業管理

優先順序3：捷運市政府站（規劃中）
• 距離：步行5分鐘
• 優勢：捷運地下空間
• 設施：完整逃生設備
• 限制：視捷運建設進度

七期特殊考量：
• 高樓建築密集，疏散時間較長
• 金融商業中心，經濟價值高
• 人潮密集，需維持疏散秩序
• 多條主要道路可快速疏散

避免區域：
• 台中市政府大樓（政治目標）
• 台中歌劇院（文化地標）
• 秋紅谷廣場（開放空間無掩護）
• 新光三越周邊（人群聚集）
```

#### 情境二：逢甲夜市商圈
```
威脅：逢甲大學及夜市遭威脅
地點：逢甲夜市、逢甲大學
人員：大學師生、夜市攤商、遊客

立即避難選擇：
優先順序1：逢甲大學體育館地下室
• 距離：校園內可達
• 優勢：大學建築結構堅固
• 容量：體育館空間大
• 管理：大學保全管理

優先順序2：西屯區公所地下室
• 距離：步行10分鐘
• 優勢：政府建築安全
• 管理：區公所管理
• 設施：基本避難設施

優先順序3：大型社區地下停車場
• 距離：就近選擇
• 優勢：住宅區地下空間
• 管理：社區管理委員會

夜市特殊風險：
• 夜間人潮密集，疏散困難
• 攤販設施可能阻礙逃生
• 外地遊客不熟悉環境
• 狹窄巷道易發生踩踏

避難注意事項：
□ 夜市攤商協助疏散指引
□ 大學師生組織化疏散
□ 外地遊客特別照顧
□ 維持疏散路線暢通
```

### 中區歷史核心

#### 情境三：台中火車站周邊
```
威脅：台中車站交通樞紐遭攻擊
地點：台中火車站、中央市場
人員：通勤族、市場攤商、老城區居民

立即避難選擇：
優先順序1：台中火車站地下空間
• 距離：就近可達
• 優勢：火車站地下空間廣闊
• 容量：可容納數千人
• 設施：車站基本設施

優先順序2：中山堂地下室
• 距離：步行5分鐘
• 優勢：文化建築結構堅固
• 歷史：日治時期重要建築
• 管理：文化局管理

優先順序3：台中教育大學地下室
• 距離：步行8分鐘
• 優勢：大學建築安全標準
• 管理：學校保全管理
• 環境：校園相對安靜

老城區特色：
• 建築老舊，結構安全需注意
• 巷弄狹窄，疏散路線有限
• 高齡人口比例較高
• 傳統市場人潮密集

避免區域：
• 台中火車站月台區（交通目標）
• 中央市場（老舊建築、人群密集）
• 台中公園開放區域（無掩護）
```

### 清水區港區

#### 情境四：台中港周邊
```
威脅：台中港軍商用設施遭攻擊
地點：台中港、清水區
人員：港區員工、漁民、附近居民

立即避難選擇：
優先順序1：清水國中、清水高中
• 距離：港區10分鐘車程
• 優勢：學校建築安全標準
• 管理：學校保全管理
• 環境：遠離港區危險範圍

優先順序2：清水區公所地下室
• 距離：步行15分鐘
• 優勢：政府建築結構堅固
• 管理：區公所管理
• 設施：基本行政設施

優先順序3：清水紫雲巖
• 距離：步行20分鐘
• 優勢：宗教建築，歷史悠久
• 管理：廟方管理
• 文化：宗教心靈慰藉

港區特殊風險：
• 軍用碼頭為首要攻擊目標
• 石化儲槽可能爆炸起火
• 海上攻擊威脅
• 疏散路線相對有限

避免區域：
• 台中港軍用碼頭5公里範圍
• 石化工業區3公里範圍
• 港區聯外道路（台17線）
• 梧棲漁港（次要目標）
```

## 🏃‍♂️ 短期避難情境範例

### 東勢區山城避難

#### 情境五：都會區往山城避難
```
背景：
威脅：台中都會區軍事威脅
地點：東勢區、和平區
人員：台中市區避難者
目標：山城短期避難3-7天

避難路線：
主要路線：台中→豐原→東勢
起點：台中市區
↓ 沿台3線北上
豐原區
↓ 繼續台3線東行
東勢區
↓ 沿台8線進山
和平區

距離：約40公里
時間：開車1.5小時，大眾運輸2.5小時

避難設施選擇：
優先選擇：東勢國小、和平國小
• 山城學校設施
• 可容納數百人
• 基本教育設施
• 客家文化環境

備用選擇：東勢林場
• 森林遊樂區設施
• 住宿條件良好
• 自然環境優美
• 可能需要付費

山城特色：
• 東勢客家文化
• 和平泰雅族部落
• 大甲溪水源
• 中橫公路起點

避難資源：
□ 大甲溪水源充足
□ 山區森林資源
□ 客家農業技術
□ 泰雅族狩獵知識

避免區域：
• 德基水庫大壩
• 台8線中橫公路主線
• 和平電廠（重要設施）
• 主要觀光景點
```

避難設施：
• 后里、神岡農會

避免區域：
