# 桃園市避難情境範例

## 概述

桃園市位於台灣西北部，擁有國際機場、高科技產業、多元族群文化。因桃園機場的國際重要性，面臨特殊的軍事威脅風險。本文件針對桃園市各區域特性，提供具體的避難情境範例與應對策略。

---

## ⚡ 立即避難情境範例

### 桃園機場周邊區域

#### 情境一：機場從業人員與旅客
```
威脅：桃園國際機場遭飛彈攻擊
地點：桃園國際機場航廈及周邊
人員：機場員工、航空公司人員、旅客

立即避難選擇：
優先順序1：機場航廈地下層
• 距離：航廈內直達
• 優勢：專業防災設計，結構最堅固
• 容量：兩航廈可容納數萬人
• 設施：完整生命維持系統、醫療設施

優先順序2：機場捷運A12、A13站
• 距離：步行5-10分鐘
• 優勢：地下捷運站結構堅固
• 容量：捷運站空間充足
• 管理：桃園捷運專業管理

優先順序3：華泰名品城地下停車場
• 距離：步行10分鐘
• 優勢：大型購物中心地下空間
• 設施：商場基本設施
• 限制：可能人潮擁擠

機場特殊風險：
• 跑道、航廈為首要攻擊目標
• 航空燃油可能引發大火爆炸
• 國際旅客語言溝通困難
• 疏散距離較遠，時間緊迫

避免區域：
• 機場跑道周邊1公里（爆炸範圍）
• 航油設施周邊500公尺（火災風險）
• 停機坪、塔台（軍事目標）
• 機場聯外道路（交通要道）
```

#### 情境二：大園、蘆竹住宅區
```
威脅：機場攻擊波及周邊住宅區
地點：大園區、蘆竹區住宅社區
人員：機場周邊居民、通勤族

立即避難選擇：
優先順序1：大園國小、蘆竹國小
• 距離：步行5-10分鐘
• 優勢：學校建築安全標準
• 管理：學校保全管理
• 環境：相對遠離機場危險區

優先順序2：社區活動中心
• 距離：步行5分鐘內
• 優勢：社區公共設施
• 管理：里長、社區管理
• 熟悉：居民熟悉環境

優先順序3：大型社區地下停車場
• 距離：就近選擇
• 優勢：新建社區結構良好
• 管理：社區管理委員會

特殊考量：
• 機場噪音區居民對警報敏感度較低
• 外籍移工比例較高，語言溝通
• 農業區疏散路線較少
• 需快速遠離機場危險範圍

避免區域：
• 桃園機場10公里範圍
• 機場聯外道路（國道2號等）
• 航空公司貨運區
• 機場週邊工業區
```

### 桃園市區核心

#### 情境三：桃園車站商圈
```
威脅：桃園市政治經濟中心遭攻擊
地點：桃園火車站、市政府周邊
人員：通勤族、購物民眾、市府員工

立即避難選擇：
優先順序1：桃園車站地下空間
• 距離：就近可達
• 優勢：火車站地下空間廣闊
• 容量：可容納數千人
• 設施：車站基本設施

優先順序2：桃園市政府地下停車場
• 距離：步行5分鐘
• 優勢：政府建築結構堅固
• 管理：市府保全管理
• 設施：政府基本設備

優先順序3：統領百貨地下樓層
• 距離：步行3分鐘
• 優勢：百貨公司地下空間
• 設施：商場設施完善
• 限制：商業設施可能管制

避難注意事項：
• 桃園車站為交通樞紐，可能成為目標
• 市政府為政治中心，威脅風險較高
• 商圈人潮密集，需注意秩序
• 多條道路可快速疏散

避免區域：
• 桃園市政府大樓（政治目標）
• 桃園車站月台區（交通樞紐）
• 中正路主要商圈（人群密集）
```

### 中壢工業區

#### 情境四：中壢工業區威脅
```
威脅：工業區遭攻擊或化學災害
地點：中壢工業區及內壢地區
人員：工廠員工、附近居民

立即避難選擇：
優先順序1：中壢國中、內壢國中
• 距離：各區就近選擇
• 優勢：學校建築安全標準
• 管理：學校保全管理
• 環境：遠離工業污染源

優先順序2：中壢車站地下空間
• 距離：步行10-15分鐘
• 優勢：火車站結構堅固
• 容量：車站空間充足
• 管理：台鐵專業管理

優先順序3：社區活動中心、里民會館
• 距離：就近選擇
• 優勢：社區公共設施
• 管理：社區自主管理

工業區特殊風險：
• 化學物質洩漏風險
• 爆炸火災可能性
• 有毒氣體擴散
• 環境污染影響

化災應變措施：
□ 注意風向，往上風處疏散
□ 使用濕毛巾掩住口鼻
□ 避免接觸不明液體
□ 快速離開污染區域

避免區域：
• 中壢工業區廠房密集區
• 化學工廠周邊1公里
• 工業區主要道路
• 污水處理廠周邊
```

## 🏃‍♂️ 短期避難情境範例

### 復興鄉原住民部落

#### 情境五：都會區往山區避難
```
背景：
威脅：桃園都會區軍事威脅
地點：復興鄉泰雅族部落
人員：桃園市區、中壢區避難者
目標：山區短期避難3-7天

避難路線：
主要路線：桃園→大溪→復興鄉
起點：桃園市區
↓ 沿台4線東行
大溪區
↓ 沿台7線進山
復興鄉
↓ 進入泰雅族部落
目的地：拉拉山、巴陵地區

距離：約50公里
時間：開車2小時，分段徒步需1天

避難設施選擇：
優先選擇：復興國小、巴陵國小
• 山區學校設施
• 可容納數百人
• 基本教育設施
• 泰雅族文化環境

備用選擇：拉拉山民宿
• 山區民宿設施
• 住宿條件較佳
• 餐飲服務完整
• 可能需要付費

部落文化交流：
• 泰雅族部落接待
• 傳統狩獵技能學習
• 山林生存智慧
• 編織手工藝體驗

避難資源：
□ 拉拉山森林資源
□ 大漢溪上游水源
□ 部落農業產品
□ 溫帶水果種植

避免區域：
• 石門水庫大壩周邊
• 台7線主要路段
• 復興區公所（政府機關）
• 主要觀光景點
```
