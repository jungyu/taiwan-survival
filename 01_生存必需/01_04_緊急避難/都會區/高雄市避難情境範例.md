# 高雄市避難情境範例

## 概述

高雄市為台灣南部最大都市，擁有重要的港口、工業區、軍事設施，同時具備山區原住民文化資源。地理從海岸延伸到中央山脈，提供多元避難選擇。本文件針對高雄市各區域特性，提供具體避難情境範例。

---

## ⚡ 立即避難情境範例

### 前金區商業核心

#### 情境一：高雄市政府周邊
```
威脅：高雄政治經濟中心遭攻擊
地點：高雄市政府、前金區商圈
人員：市府員工、金融業人士、購物民眾

立即避難選擇：
優先順序1：高雄市政府地下停車場
• 距離：步行1-2分鐘
• 優勢：政府建築結構堅固
• 容量：大型地下空間
• 管理：市府保全管理

優先順序2：捷運中央公園站
• 距離：步行3-5分鐘
• 優勢：捷運地下空間，通風良好
• 容量：捷運站空間充足
• 設施：完整逃生設備

優先順序3：新光三越高雄店地下樓層
• 距離：步行3分鐘
• 優勢：大型百貨地下商場
• 設施：商場基本設施
• 限制：可能人潮密集

商業區特殊考量：
• 金融商業中心經濟價值高
• 愛河沿岸高樓建築密集
• 人潮密集需維持秩序
• 多條道路可快速疏散

避免區域：
• 高雄市政府大樓（政治目標）
• 85大樓（地標建築）
• 愛河沿岸開放空間（無掩護）
• 中山路商圈（人群密集）
```

#### 情境二：高雄車站商圈
```
威脅：高雄車站交通樞紐遭攻擊
地點：高雄火車站、三民區商圈
人員：通勤族、購物民眾、旅客

立即避難選擇：
優先順序1：高雄車站地下空間
• 距離：就近可達
• 優勢：火車站地下空間廣闊
• 容量：可容納數千人
• 設施：車站基本設施

優先順序2：捷運高雄車站
• 距離：站內直達
• 優勢：捷運地下空間深層
• 設施：完整捷運設施
• 管理：高雄捷運管理

優先順序3：高雄中學地下室
• 距離：步行10分鐘
• 優勢：學校建築安全標準
• 管理：學校保全管理
• 環境：校園相對安靜

避難注意事項：
• 高雄車站為南台灣重要交通樞紐
• 建國路商圈人潮密集
• 三民區住宅密度高
• 需快速疏散至安全區域

避免區域：
• 高雄火車站月台區（交通目標）
• 建國路商圈（人群聚集）
• 九如路主要幹道（交通要道）
```

## 🏃‍♂️ 短期避難情境範例

### 旗山、美濃客家山城

#### 情境五：都會區往客家山城避難
```
背景：
威脅：高雄都會區軍事威脅
地點：旗山區、美濃區、六龜區
人員：高雄市區避難者
目標：客家山城短期避難3-7天

避難路線：
主要路線：高雄→旗山→美濃
起點：高雄市區
↓ 沿台21線北上
旗山區
↓ 沿台28線東行
美濃區
↓ 沿台27線往山區
六龜區

距離：約60公里
時間：開車2小時，大眾運輸3小時

避難設施選擇：
優先選擇：旗山國小、美濃國小
• 客家山城學校
• 可容納數百人
• 基本教育設施
• 客家文化環境

備用選擇：美濃客家文物館
• 文化設施空間
• 客家文化保存
• 教育展示功能
• 政府管理維護

客家山城特色：
• 旗山香蕉王國
• 美濃菸樓文化
• 六龜溫泉資源
• 客家傳統聚落

避難資源：
□ 荖濃溪水系水源
□ 客家農業技術
□ 山區溫泉資源
□ 客家文化學習

避免區域：
• 旗山老街（觀光人群聚集）
• 台21線主要路段
• 重要水利設施
• 大型觀光景點
```

背景：

避難優勢：
