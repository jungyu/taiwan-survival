# 台北市避難情境範例

## 概述

台北市作為政治、經濟、文化中心，面臨最高威脅風險，但同時擁有最完善的基礎設施。本文件針對台北市各行政區，提供具體的避難情境範例與應對策略。

---

## ⚡ 立即避難情境範例

### 中正區政治核心區

#### 情境一：總統府周邊上班族
```
威脅：政府核心區遭飛彈攻擊
地點：重慶南路一段政府機關
人員：公務員及附近上班族

立即避難選擇：
優先順序1：台大醫院地下室
• 距離：步行3-5分鐘
• 優勢：醫院建築堅固，醫療資源
• 容量：地下空間大，可容納千人
• 管理：醫院24小時有人管理

優先順序2：捷運台大醫院站
• 距離：步行2-3分鐘
• 優勢：地下深層結構，防護佳
• 設施：完整逃生設備
• 限制：容量相對有限

優先順序3：二二八和平公園地下停車場
• 距離：步行5分鐘
• 優勢：大型地下空間
• 管理：公園管理處管理

避免區域：
• 總統府周邊300公尺（首要攻擊目標）
• 立法院、司法院周邊200公尺
• 中正紀念堂廣場（開放空間無掩護）
• 台北車站周邊（交通樞紐目標）
```

#### 情境二：西門町商圈遊客
```
威脅：台北車站遭攻擊，波及西門町
地點：西門町步行街
人員：國內外遊客、年輕人

立即避難選擇：
優先順序1：西門捷運站地下空間
• 距離：就近可達
• 優勢：捷運站結構堅固
• 容量：板南線大站，空間充足
• 管理：台北捷運管理

優先順序2：新光三越台北站前店地下室
• 距離：步行3分鐘
• 優勢：大型百貨地下空間
• 設施：商場基本設施
• 限制：人潮可能擁擠

優先順序3：成功高中地下室
• 距離：步行8分鐘
• 優勢：學校建築安全標準高
• 管理：學校保全管理

特別注意：
• 外國遊客語言溝通問題
• 人潮恐慌管制
• 步行街狹窄，注意踩踏
```

### 信義區金融商業區

#### 情境三：101大樓辦公大樓
```
威脅：信義計畫區高價值目標遭攻擊
地點：台北101及周邊辦公大樓
人員：金融業、科技業上班族

立即避難選擇：
優先順序1：台北101地下避難空間
• 距離：樓內直達
• 優勢：專業防災設計，結構最堅固
• 容量：可容納數萬人
• 設施：完整生命維持系統

優先順序2：捷運市政府站
• 距離：步行2-3分鐘
• 優勢：地下RC結構，防爆能力強
• 容量：可容納數千人
• 設施：通風、照明、通訊設備

優先順序3：信義國小地下室
• 距離：步行5分鐘
• 優勢：學校建築結構堅固
• 管理：24小時警衛管理

高風險區域：
• 台北101大樓（經濟象徵目標）
• 市政府大樓（政治目標）
• 新光三越信義店（人群聚集）
• 信義計畫區各大樓（經濟目標）

避難注意事項：
• 高樓疏散時間較長
• 電梯可能停止運作
• 樓梯疏散需要秩序
• 攜帶公司重要資料
```

人員：師生、研究人員、附近居民

立即避難選擇：
優先順序1：台大圖書館地下室

• 優勢：建築結構堅固，空間大
• 設施：基本照明、通風設備
• 管理：校園保全管理

優先順序2：台大醫院地下空間
• 距離：步行5-8分鐘
• 優勢：醫院防災設計，醫療資源
• 容量：大型醫院地下空間
• 醫療：緊急醫療支援

優先順序3：大安森林公園地下停車場
• 距離：步行10分鐘
• 優勢：大型地下空間
• 環境：綠地環境，空氣相對清新

校園特殊考量：
• 實驗室危險物質處理

```
威脅：松山機場軍民用設施遭攻擊
地點：松山機場周邊住宅區
人員：機場工作人員、附近居民

立即避難選擇：
優先順序1：捷運松山機場站地下空間
• 距離：步行3-5分鐘
• 優勢：機場捷運站，結構堅固
• 設施：完整逃生設備
• 限制：可能成為攻擊目標

優先順序2：敦化國小地下室
• 距離：步行8分鐘
• 優勢：學校建築安全標準
• 管理：學校保全管理
• 環境：遠離機場危險區

優先順序3：民生社區地下停車場

• 松山機場跑道周邊2公里
• 航空站大廈（民用目標）
• 空軍基地（軍事目標）
• 濱江街飛機觀景區

特殊注意：
• 機場可能有油料爆炸風險
• 飛機可能緊急降落
• 空域管制，直升機救援困難
```

## 🏃‍♂️ 短期避難情境範例

### 文山區往新店山區

#### 情境六：政大學區避難
```
背景：
威脅：台北盆地南區軍事活動
地點：政治大學及指南山區
人員：政大師生、木柵居民
目標：新店山區短期避難

避難路線：
主要路線：政大→新店→烏來
起點：政治大學
↓ 沿指南路下山
木柵市區
↓ 經木新路
新店市區
↓ 沿新烏路上山
烏來風景區

距離：約25公里
時間：開車1.5小時，大眾運輸2小時

避難地點選擇：
優先選擇：烏來國中小
• 教育設施，空間充足
• 山區環境，空氣清新
• 泰雅族部落文化交流

備用選擇：烏來溫泉民宿
• 完整住宿設施
• 溫泉療養功能
• 可能需要付費

避難特色：
• 政大國際師生多元化
• 指南山區地形掩護
• 烏來原住民文化學習
• 溫泉資源身心療癒

避免區域：
• 木柵動物園（觀光目標）
• 政大包種茶研究中心（研究設施）
• 新店軍事學校（軍事目標）
```

## 🏔️ 長期避難情境範例

### 全市疏散往宜蘭山區

#### 情境八：台北市全面疏散
```
背景：
威脅：台北都會區遭全面攻擊
人口：需疏散200萬人以上
目標：宜蘭縣山區建立避難社區

大規模疏散組織：
第一波：政府官員、軍警、醫護
• 優先確保政府功能
• 維持基本治安醫療
• 建立疏散指揮體系

第二波：老弱婦孺、學生
• 學校統一組織疏散
• 醫院病患轉移
• 社福機構集體疏散

第三波：一般民眾
• 依行政區分批疏散
• 企業組織員工疏散
• 自主避難者

疏散路線分流：
北線：士林、北投→陽明山→金山→基隆→宜蘭
中線：中正、大同、萬華→新北→雪山隧道→宜蘭
南線：大安、信義、松山→新店→坪林→宜蘭
東線：內湖、南港→汐止→平溪→雙溪→宜蘭

宜蘭山區避難社區建立：
大同鄉泰雅部落社區：
• 收容人口：10萬人
• 主要來源：北投、士林、內湖區民
• 部落整合：寒溪、四季、英士部落
• 產業發展：溫泉觀光、高山農業

員山鄉農業社區：
• 收容人口：15萬人
• 主要來源：中正、大同、萬華區民
• 農業基礎：稻米、蔬菜、水果
• 產業發展：有機農業、農產加工

南澳鄉原住民社區：
• 收容人口：8萬人
• 主要來源：大安、信義、松山區民
• 文化融合：泰雅族獵人文化
• 產業發展：生態旅遊、原住民工藝

社區組織架構：
□ 區域指揮中心：統籌各社區協調
□ 社區自治會：日常事務管理
□ 生產合作社：農業、手工業生產
□ 教育委員會：臨時學校設立
□ 醫療委員會：醫療資源整合
□ 文化委員會：精神文化建設

避免區域：
• 蘇澳港軍用碼頭
• 宜蘭機場周邊
• 主要水庫大壩
• 重要通訊設施
```
