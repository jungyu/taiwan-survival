以下為針對「緊急避難儲糧」設計的 **每人三日配量建議表**，依據三種常見避難情境（個人背包攜行、車載儲備、在家避難），提供合理份量、品項組合與每日攝取建議，考量因素包括：

* 每日熱量建議：成人最低需求約 1,600–2,000 kcal/day
* 台灣氣候條件（溫熱潮濕）
* 實際市售包裝與可攜便利性
* 是否需要炊具、飲水資源

## 📦 合理儲糧量建議（每人 3 日最低）

| 品項類型    | 最低建議數量 | 說明          |
| ------- | ------ | ----------- |
| 主食（飯、麵） | 3～6 包  | 每日1～2餐      |
| 配菜／罐頭   | 3～6 罐  | 提供蛋白質、風味    |
| 零食／補給   | 6～9 包  | 快速補能、口感變化   |
| 飲品      | 3～6 瓶  | 可換算為水源      |
| 飲水      | ≥6 公升  | 飲用＋簡易煮食用    |
| 沖泡品／湯品  | 3～6 包  | 可作開水來源、熱量補充 |
