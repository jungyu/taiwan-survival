# 個人隨身醫藥器材清單

## 概述

個人隨身醫藥器材包（IFAK - Individual First Aid Kit）是每個人都應該隨身攜帶的基本醫療用品，主要用於處理常見外傷、緊急止血和基本生命支持。此清單依照使用頻率、攜帶便利性和緊急重要性進行分類。

---

## 🚨 核心急救物品（優先級A）

### 止血控制用品
```
□ 戰術止血帶 × 1
  ・型號建議：C-A-T或SOF-T Wide
  ・用途：控制四肢大出血
  ・重量：約100-150g

□ 止血紗布 × 2包
  ・建議：QuikClot Combat Gauze 或 Celox Rapid
  ・規格：3英寸×4碼
  ・用途：深度傷口快速止血

□ 壓迫繃帶 × 1
  ・建議：以色列繃帶（Israeli Bandage）
  ・規格：4英寸
  ・特點：內建壓力棒，單手操作

□ 醫用膠帶 × 1卷
  ・規格：2.5cm×5m
  ・材質：透氣膠布
  ・用途：固定敷料
```

  ・用途：剪開衣物、繃帶

## 📦 包裝與攜帶

### 容器選擇
```
建議規格：
□ 硬殼收納盒（防水、防壓）
□ 尺寸：20×15×8cm
□ 重量限制：1-1.5kg
□ 透明隔層便於取用

替代方案：
□ 戰術腰包
□ 登山腰包
□ 多功能工具包
```

### 分類包裝
```
區域A - 緊急用品：
• 止血帶、止血紗布
• 壓迫繃帶、手套

區域B - 一般處理：
• 紗布、繃帶、膠帶
• 消毒用品、藥膏

區域C - 藥品：
• 口服藥品（密封袋）
• 外用藥品

區域D - 工具：
• 剪刀、鑷子、手電筒
```
