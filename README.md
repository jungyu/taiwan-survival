## 📖 專案簡介

這是一個專為台灣環境設計的戰時與緊急情況生存指南，整合了國際衝突經驗、人道主義標準，以及台灣地理文化特色。涵蓋從平時準備到戰時求生的完整知識體系。

### 🎯 核心特色

- **🌍 國際經驗融合**：整合烏克蘭、以巴、敘利亞等衝突的實戰經驗
- **🏝️ 台灣環境優化**：針對台灣地理、氣候、文化特色量身打造
- **📱 離線完備**：所有關鍵資源都支援離線使用
- **🎯 分級應對**：從0-72小時到長期重建的完整時間軸
- **🔬 科學依據**：基於WHO、UNHCR、Sphere等國際標準

## 🎓 學習路徑

### 👨‍👩‍👧‍👦 家庭使用者

1. **第一週**：閱讀基礎準備指南，制定家庭應變計劃
2. **第二週**：建立應急物資儲備，學習基本急救

4. **第四週**：參與社區活動，建立支援網絡

2. **進階管理**：避難處所管理與資源協調
3. **專業技能**：TCCC急救與危機溝通
4. **長期規劃**：永續生活與社區重建

### 👨‍⚕️ 醫護人員

1. **戰術急救**：TCCC訓練與實戰應用
2. **大量傷患**：災難醫學與檢傷分類
3. **資源管理**：醫療物資配置與循環
4. **心理支援**：創傷後壓力與心理急救
